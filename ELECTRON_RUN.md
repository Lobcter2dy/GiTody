# Как запустить Electron версию

## Способ 1: Прямой запуск (рекомендуется)

```bash
npm run electron:dev
```

Эта команда:
1. Запустит Vite dev сервер (http://localhost:5173)
2. Подождет пока сервер запустится
3. Запустит Electron приложение

## Способ 2: Запуск в два этапа

### Шаг 1: Запустите веб-сервер в одном терминале
```bash
npm run dev
```

### Шаг 2: В другом терминале запустите Electron
```bash
npx electron .
```

## Способ 3: Сборка и запуск готовой версии

### Сначала соберите приложение:
```bash
npm run build
```

### Затем запустите Electron:
```bash
NODE_ENV=production npx electron .
```

## Если не работает:

1. **Убедитесь что установлены зависимости:**
```bash
npm install
```

2. **Проверьте что Electron установлен:**
```bash
npm list electron
```

3. **Установите Electron вручную:**
```bash
npm install --save-dev electron
```

4. **Проверьте конфигурацию в package.json:**
```json
{
  "main": "electron/main.js"
}
```

## Ожидаемый результат:

После запуска `npm run electron:dev` должно:
- Открыться окно Electron приложения
- Загрузиться интерфейс GitHub Manager
- В консоли (если откроете DevTools) не должно быть ошибок

## Отладка:

Если приложение не запускается, проверьте консоль терминала где запустили команду - там будут показаны ошибки.
